데이터의 기초

Lesson 1-1 데이터란? · 1-2 데이터 유형 · 1-3 데이터 품질
1-1 · 데이터란 무엇인가?
데이터(Data)
관찰·측정으로 수집된 정보.
측정 가능 · 기록 가능 · 분석 가능한 것만 데이터다.
데이터의 3가지 특성
① 측정 가능 — 일관된 기준으로 기록 가능
② 기록 가능 — 누구나 같은 값 확인 가능
③ 분석 가능 — 패턴·비교·예측에 활용 가능
⚠️
데이터 vs 의견 구별법
"저 가수 노래가 진짜 좋아" → 주관적 의견 (❌데이터 아님)
"이 노래 스트리밍 횟수 320만 회" → 측정·기록된 수치 (✅데이터)
1-2 · 데이터 유형
유형특징예시평균?
범주형
Categorical		 이름·그룹으로 분류
계산 불가		 혈액형, 성별, 좋아하는 색, 우편번호 의미 없음
수치형
Numerical		 숫자로 표현
사칙연산 가능		 키, 체온, 독서 시간, 월 소득 의미 있음
💡
핵심 판단 기준 — "평균을 내면 의미가 있는가?"
우편번호 03172의 평균 → 의미 없음 → 범주형
버스 노선 번호(472번, 100번)도 숫자지만 → 범주형 (평균 노선 번호는 무의미)
📐
더 세밀한 분류 (4단계 척도)
명목 → 순서 → 등간 → 비율 순으로 정밀도 증가
명목: 혈액형 · 순서: 만족도 · 등간/비율: 키·온도·점수
1-3 · 데이터 품질 (5가지 기준)
기준의미위반 예시
완전성결측값 없음몸무게 칸이 비어있음
정확성현실적으로 가능한 값몸무게 -5kg
일관성동일한 기준으로 기록날짜: 2026/01/01 vs Jan 2 vs 1월 3
최신성분석 목적에 맞게 최신2010년 물가로 2026년 분석
중복 없음같은 데이터 반복 없음동일 학생 행이 두 번 입력
🗑️
GIGO 원칙 (Garbage In, Garbage Out)
입력 데이터가 나쁘면 아무리 좋은 분석 도구를 써도 결과는 나쁘다.

데이터 질문 설계

Lesson 2-1 질문 변환 · 2-2 실제 시나리오 · 2-3 질문 품질 평가
좋은 데이터 질문의 4가지 조건
1
측정 가능한 변수 포함 "하루 독서 시간(분)", "어휘력 점수"처럼 구체적으로 측정 가능해야 한다
2
분석 대상 명확 "고등학생", "서울 시민"처럼 누구를 분석하는지 명시
3
데이터로 답 가능 주관적 판단이나 가치 판단이 아닌, 데이터로 확인할 수 있어야 한다
4
중립적 표현 결론을 미리 가정하거나 특정 답을 유도하지 않는다
❌ 나쁜 질문 예시
  • "SNS는 해로운가?" → 가치판단
  • "왜 요즘 학생들은 책을 안 읽을까?" → why 질문(인과 가정)
  • "기후 변화가 심각한가?" → 기준 없는 주관적 판단
  • "수면을 충분히 취하면 성적이 오르는가?" → '충분히' 기준 없음
✅ 좋은 질문 예시
  • "고등학생의 하루 독서 시간과 어휘력 점수 사이에 양의 상관관계가 있는가?"
  • "주 4회 이상 운동하는 학생의 체력 점수 평균이 그렇지 않은 학생보다 높은가?"
  • "2016~2025년 한국 청소년의 평균 수면 시간은 감소 추세를 보이는가?"
설문 설계 — 흔한 오류 유형
오류 유형예시개선
유도 질문 "우리 학교 도서관이 불편하다고 느끼지 않으신가요?" "우리 학교 도서관에 대해 어떻게 생각하세요?"
이중 질문 "음악을 좋아하고 악기를 배우고 싶으신가요?" 두 가지 질문으로 분리
모호한 표현 "자주 독서하시나요?" "한 달에 몇 권의 책을 읽으시나요?"
사회적 압력 "많은 전문가가 권장하는 채식 식단에 관심이 있으신가요?" "채식 식단에 대해 어떻게 생각하세요?"
📝 시험 포인트
  • 유도 질문: 질문 자체에 방향이 내포된 경우
  • 이중 질문: 한 문장에 두 가지 이상을 동시에 묻는 경우
  • "많이/자주/별로" 같은 기준 없는 표현 → 측정 불가 → 나쁜 질문

표본과 편향

Lesson 3-1 모집단 vs 표본 · 3-2 편향의 종류 · 3-3 설문 설계
3-1 · 모집단 vs 표본
모집단 (Population)
분석하고자 하는 전체 집단
예: 전국 고등학생 130만 명
표본 (Sample)
실제 조사한 일부 집단
예: 무작위 추출한 1,000명
🚨
표본의 두 가지 핵심 조건
1. 크기 — 클수록 안정적 (큰 수의 법칙)
2. 추출 방법 — 무작위(랜덤) 추출이어야 대표성 확보
⚠️ 크기만 크다고 대표성이 보장되지 않는다!
💡
온라인 커뮤니티 찬반투표의 함정
참여자가 수만 명이어도, 해당 커뮤니티에 관심 있는 사람들만 자발적으로 참여 → 선택 편향 발생 → 신뢰 불가
3-2 · 3가지 편향 완전 정리
편향 종류정의대표 사례키워드
선택 편향
Selection Bias		 특정 집단만 포함되어 대표성 부족 도서관 이용 만족도 조사 → 도서관 방문자만 응답
온라인 커뮤니티 투표 → 해당 커뮤니티 회원만 참여		 대표성 없음
생존자 편향
Survivorship Bias		 성공·생존한 것만 보이고 실패·소멸은 안 보임 유명 작가 인터뷰로 "글쓰기 비결" 연구 → 출판된 작가만 대상
오랜 기업의 경영 방식 연구 → 살아남은 기업만 분석		 실패자 누락
확인 편향
Confirmation Bias		 기존 믿음을 확인하는 정보만 선택적으로 수집 특정 학습법 효과를 믿음 → 성공 사례만 검색
특정 식품이 몸에 좋다고 믿음 → 긍정 연구만 수집		 선택적 수집
📖
생존자 편향 — 유명 작가 사례
출판된 유명 작가 50명을 인터뷰 → "매일 2시간 글쓰기가 성공 비결"이라는 결론
→ 같은 습관으로 글을 썼지만 출판에 실패한 수천 명은 인터뷰 대상에 없다
→ 데이터에 잡히지 않은 '실패자'를 고려해야 올바른 결론을 낼 수 있다

데이터 탐색

Lesson 4-1 테이블 읽기 · 4-2 기술통계 · 4-3 패턴 발견 · 4-4 이상치
4-1 · 데이터 테이블 구조
행 (Row)
하나의 관측 단위
예: 학생 1명의 전체 정보
열 (Column)
하나의 변수
예: 모든 학생의 키
셀 (Cell)
특정 관측치의 특정 변수값
예: 학생1의 키 = 172cm
데이터 탐색 5단계 체크리스트
1
크기 확인행 × 열 수 확인 (데이터 규모 파악)
2
열 이름·의미 확인각 변수가 무엇을 의미하는지
3
데이터 유형 파악수치형 vs 범주형 구분
4
기술 통계 확인평균, 중앙값, 표준편차, 최솟값, 최댓값
5
이상값·결측값 확인비현실적인 값, 빈 값 탐색
4-2 · 기술통계 — 평균 vs 중앙값 vs 표준편차
통계량정의장점단점언제 사용?
평균
(Mean)		 모든 값의 합 ÷ 개수 직관적, 계산 쉬움 이상치에 민감하게 왜곡 이상치 없는 대칭 분포
예: 시험점수, 키
중앙값
(Median)		 정렬 후 가운데 값 이상치에 강건 총합 파악 불가 이상치 있는 치우친 분포
예: 소득, 부동산 가격
표준편차
(Std Dev)		 평균으로부터의
평균적 거리		 데이터 퍼짐 파악 이상치 영향 받음 두 집단 분포 비교
예: 반별 성적 편차 비교
💡
평균 vs 중앙값 크기 비교로 분포 유추
평균 > 중앙값 → 오른쪽에 큰 값 존재 (우측 꼬리 분포)
평균 ≈ 중앙값 → 대칭 분포
평균 < 중앙값 → 왼쪽에 작은 값 존재 (좌측 꼬리 분포)
📊
같은 평균, 다른 표준편차
1반: 평균 82점, SD=3 → 80~84점 사이에 밀집 (성적 고름)
2반: 평균 82점, SD=19 → 50~100점 넓게 분포 (성적 들쭉날쭉)
4-4 · 이상치 (Outlier)
// IQR 이상치 탐지 공식
IQR = Q3 - Q1
하한 = Q1 - 1.5 × IQR ← 이보다 작으면 이상치
상한 = Q3 + 1.5 × IQR ← 이보다 크면 이상치
🚨
이상치 처리 원칙 — "절대 무조건 삭제 금지"
1단계: 이상치 발생 원인 파악
2단계: 데이터 오류인지 실제 값인지 판단
3단계: 처리 방법 결정 (삭제/별도 분석/전문가 확인)
4단계: 처리 방법을 투명하게 보고

6-1 · 상관계수 r
-1
강한 음
중간 음

0 (무관)
약한 양
중간 양

절댓값 범위해석예시 r값
|r| > 0.7강한 상관관계r = 0.89, r = -0.75
0.4 < |r| ≤ 0.7중간 상관관계r = 0.58, r = -0.62
0.2 < |r| ≤ 0.4약한 상관관계r = 0.31

🚨
상관계수 강도는 절댓값으로 비교!
r = -0.75 vs r = 0.62 → |-0.75| = 0.75 > 0.62 → -0.75가 더 강한 상관관계
6-2 · 상관관계 vs 인과관계 (가장 중요!)
상관관계 (Correlation)
  • 두 변수가 함께 변하는 패턴
  • r값으로 측정 가능
  • 관찰만으로 확인 가능
  • 원인-결과 아님
인과관계 (Causation)
  • X가 실제로 Y를 일으킴
  • 실험·통제 연구 필요
  • 단순 관찰로 증명 불가
  • 시간적 선행 필요 (원인이 먼저)
❄️
핵심 원칙 — "상관관계 ≠ 인과관계"
r이 아무리 높아도 인과관계를 보장하지 않는다.

심슨의 역설 (Simpson's Paradox)
심슨의 역설은 통계에서 아주 유명하고 재미있는 착시 현상입니다. 한마디로 요약하면 전체 평균을 볼 때와 그룹별로 쪼개서 볼 때의 결과가 완전히 뒤바뀌거나, 겉으로 보이는 순위가 달라지는 현상을 말합니다.

이름은 거창하지만, 예시를 보면 금방 이해할 수 있습니다. 가장 널리 쓰이는 「어느 병원에 가야 할까?」 예시로 설명해 보겠습니다.
🏥 예시: A병원과 B병원, 어디가 더 수술을 잘할까?
수술을 받아야 해서 두 병원의 최근 수술 생존율을 찾아봤다고 가정해 봅시다. 아래 숫자는 전체·중증·경증이 서로 합이 맞도록 맞춘 예시입니다. (각 병원 모두 수술 1,000명 기준)
구분전체 생존율생존 인원 / 전체
A 병원82%820명 / 1,000명
B 병원95%950명 / 1,000명
전체 통계만 보면 B 병원 생존율이 압도적으로 높습니다. 그런데 환자를 중증(수술이 어려운 환자)과 경증(수술이 쉬운 환자)으로 나눠 보면 이야기가 달라집니다.
🕵️ 환자 상태에 따른 생존율 비교
병원중증 (생존/전체)중증 생존율경증 (생존/전체)경증 생존율검산 (중증+경증)
A 715 / 894명 79.98% 105 / 106명 99.06% 715+105 = 820명 ✓
B 75 / 107명 70.09% 875 / 893명 97.98% 75+875 = 950명 ✓
인원 합계: A는 894+106 = 1,000명, B는 107+893 = 1,000명 · 전체 생존율 820÷1,000 = 82%, 950÷1,000 = 95%
1. 중증 환자
79.98% (A) > 70.09% (B) → A 병원이 더 잘 살렸습니다.
2. 경증 환자
99.06% (A) > 97.98% (B) → A 병원이 더 잘 살렸습니다.
🤔
이상하지 않나요?
중증·경증 모든 유형에서 A 병원 생존율이 더 높은데, 전체 1,000명 기준으로는 B(95%)가 A(82%)보다 훨씬 높게 보입니다. 환자 구성(중증 비율)이 다르기 때문입니다.
💡 왜 이런 역설이 발생할까요?
비밀은 환자의 비율, 즉 데이터 안에 숨은 집단 구성(중증도)이라는 변수에 있습니다.
  • A 병원은 큰 대학병원이라 수술이 어렵고 사망 위험이 큰 중증 환자 비율이 훨씬 높습니다. 아무리 수술을 잘해도 어려운 케이스가 많이 몰리면 전체 생존율 평균이 깎여 보입니다.
  • B 병원은 작은 동네 병원이라 경증 환자가 대부분입니다. 원래 살 확률이 높은 환자만 많이 수술하면 전체 생존율이 올라가 보입니다.
🔀
정리하면 심슨의 역설이란
이렇게 숨겨진 집단(중증도 등)을 무시한 채 전체 합계·평균만 뭉뚱그려 비교하면 잘못된 결론에 빠질 수 있다는 뜻입니다. → 항상 집단별로도 나누어 분석해야 합니다.
📊 결론 및 시사점
심슨의 역설이 주는 교훈은 분명합니다. 전체 평균·만족도·합격률만 보고 섣불리 결론 내리면 안 된다는 것입니다.

뉴스나 광고에서 「우리 제품 전체 만족도 90%!」「이 학교 평균 합격률이 최고!」처럼 말할 때, 그 이면을 연령·지역·난이도·상황 등으로 나누어 보면 전혀 다른 그림이 나올 수 있다는 점을 항상 염두에 두어야 합니다.
6-4 · 분석 결과 검증 5단계
1
수치 확인 (Numbers Check) 데이터 크기와 통계량이 정확한가?
2
시각화 확인 (Visualization Check) 차트가 데이터를 정확하게 반영하고 있는가?
3
상식 확인 (Sense Check) 결과가 현실적으로 말이 되는가? 도메인 지식과 일치하는가?
4
민감도 분석 (Sensitivity Analysis) 이상치를 제거하거나 기간을 바꿔도 결론이 유지되는가?
5
대안 분석 (Alternative Analysis) 다른 분석 방법을 써도 같은 결론이 나오는가?
📝 시험 포인트 — 각 단계의 핵심 질문
  • 민감도 분석: "이상치를 제거해도 결론이 유지되는가?"
  • 대안 분석: "다른 집단별로 나눠도 같은 경향인가?"
  • 상식 확인: "이 결과가 현실에서 납득 가능한가?"
