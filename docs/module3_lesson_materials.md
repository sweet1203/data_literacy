## 3-1 표본과 모집단

### 핵심 개념
- 모집단: 알고 싶은 전체 대상 (예: 전국 고등학생 130만 명)
- 표본: 실제로 조사한 일부 (예: 1,000명)
- 표본으로 모집단의 특성을 추정한다.

### 전수조사를 하지 않는 이유
| 이유 | 설명 |
|------|------|
| 비용 | 130만 명 설문에 수십억 원 필요 |
| 시간 | 전수조사에 몇 달–몇 년 소요 |
| 현실성 | 모든 대상에게 접근 자체가 불가능한 경우 있음 |

- 인구주택총조사는 5년마다 전수조사하지만 비용이 수천억 원. 일반 여론조사는 1,000–3,000명으로 전체 추정.

### 표본 크기가 중요한 이유
- 표본이 작으면 결과가 크게 흔들림. 커질수록 실제 비율에 가까워짐 → 큰 수의 법칙.
- 5개 뽑기: 매우 불안정 (20%–100% 변동)
- 30개: 어느 정도 안정 (±10% 내외)
- 100개: 매우 안정 (±5% 내외)

### 조사 결과 믿을 수 있을까? (예시)
- "전국 고등학생 10명에게 물어본 결과 80%가 교복에 불만" → 주의 필요 (표본 10명은 너무 적음)
- "1,500명 무작위 전화조사, 대통령 지지율 52%" → 믿을 수 있음
- "인스타 투표 5만 명 참여, 90% 찬성" → 주의 필요 (자기선택 편향)
- "학교 전교생 800명 전수조사, 급식 만족도 72%" → 믿을 수 있음 (해당 학교에 대해서는 정확)

### 유의
- 표본 크기뿐 아니라 뽑는 방식(무작위인지, 편향되었는지)도 중요. 다음 레슨에서 편향 학습.

## 3-2 편향의 세계

### 도입
- 데이터에 보이지 않는 편향(Bias)이 숨어 있을 수 있음. 편향된 데이터로 분석하면 잘못된 결론에 도달.

### 3가지 주요 편향

| 편향 | 영문 | 설명 | 예시 |
|------|------|------|------|
| 선택 편향 | Selection Bias | 특정 집단만 데이터에 포함되어 전체를 대표하지 못함 | 학교 앞에서만 설문→학생만 응답 / 온라인 조사→인터넷 사용자만 |
| 생존자 편향 | Survivorship Bias | 살아남은(성공한) 것만 보고, 실패·탈락한 경우를 무시 | 성공한 중퇴자만 보고 "중퇴해도 성공한다" 결론 |
| 확인 편향 | Confirmation Bias | 기존 믿음을 확인하는 정보만 선택적으로 받아들임 | 'A 다이어트 좋다'고 믿으면 효과 후기만 눈에 들어옴 |

### 2차 대전 비행기 이야기 (생존자 편향)
- 귀환한 폭격기의 총탄 자국: 날개·동체에 많고, 엔진·조종석에 적음.
- 잘못된 판단: 총탄 자국이 많은 부위를 보강 → 올바른 판단: 자국이 적은 엔진·조종석을 보강 (그 부위에 맞은 비행기는 돌아오지 못함). 통계학자 아브라함 발드의 분석. "보이는 데이터"에만 집중하면 "보이지 않는 데이터"를 놓침.

### 뉴스/데이터 편향 판단 예시
- "아침형 인간이 성공한다 — 유명 CEO 20명 인터뷰" → 편향 있음 (생존자 편향)
- "국민 62% 찬성 — 전국 2,000명 무작위 전화조사" → 괜찮음
- "청소년 SNS 사용시간 증가 — 인스타 설문 3만 명" → 편향 있음 (선택 편향, SNS 사용자만)

### 완벽히 편향 없는 데이터?
- 현실적으로 모든 편향 제거는 어렵다. 중요한 것: (1) 편향이 있을 수 있다는 인식, (2) 어떤 편향인지 식별, (3) 결론에 영향을 미치는 정도 판단.

## 3-3 좋은 설문 vs 나쁜 설문

### 유도 질문의 함정
- 질문 속에 특정 방향의 답을 암시하면 응답자가 무의식적으로 그 방향으로 답함.
- 유도 질문: "이 비효율적인 정책에 대해 어떻게 생각하세요?" → "비효율적"이 이미 포함
- 중립적 질문: "이 정책에 대해 어떻게 생각하세요?"

### 설문 설계의 흔한 실수
| 실수 | 설명 | 예시 |
|------|------|------|
| 이중 질문 | 한 질문에 두 가지를 묻는 것 | "운동을 좋아하고 건강에 관심 있나요?" |
| 부적절한 척도 | 예/아니오만 제공하거나 척도가 균형 맞지 않음 | "만족" "매우 만족" "극도로 만족" (불만족 없음) |
| 순서 효과 | 질문 순서가 이후 응답에 영향 | 먼저 부정적 사건을 물으면 만족도가 낮아짐 |
| 모호한 기준 | 응답 기준이 사람마다 다르게 해석됨 | "자주", "가끔" |

### 좋은 설문의 5가지 원칙
1. 중립적으로 질문한다 (유도하지 않는다)
2. 한 번에 하나만 묻는다 (이중 질문 금지)
3. 균형 잡힌 척도를 사용한다 (긍정-부정 동일하게)
4. 구체적 기준을 제시한다 ("가끔" → "주 1-2회")
5. 순서 효과를 고려하여 배치한다

### 유도 vs 중립 예시
- "우리 학교 급식이 맛없다고 생각하지 않나요?" → 유도 → "우리 학교 급식의 맛에 대해 어떻게 생각하나요?"
- "대부분의 학생이 찬성하는 교복 자율화에 대해 어떻게 생각하세요?" → 사회적 압력 → "교복 자율화에 대해 어떻게 생각하세요?"

## 챕터 3 종합: 이번에 배운 것

- 3-1: 모집단과 표본의 차이, 전수조사가 어려운 이유, 표본 크기가 클수록 추정이 안정됨(큰 수의 법칙), 뽑는 방식도 중요.
- 3-2: 선택 편향(특정 집단만 포함), 생존자 편향(성공/생존한 경우만 관찰), 확인 편향(기존 믿음 확인 정보만 수용), 편향된 데이터→잘못된 결론.
- 3-3: 유도 질문이 데이터를 왜곡하는 방식, 설문 설계 흔한 실수 4가지, 좋은 설문 5원칙, 나쁜 질문 개선 실습.

## 레슨 과제

1. '모집단'과 '표본'의 차이를 한 문장씩 설명하고, 여론조사에서 전국 국민 대신 1,000명만 조사하는 이유를 적어보세요.

2. '선택 편향', '생존자 편향', '확인 편향' 중 하나를 골라, 그 편향이 들어 있을 수 있는 뉴스나 광고 예를 들고, 왜 그런지 설명해 보세요.

3. 2차 대전 때 귀환한 폭격기의 총탄 자국을 조사한 '생존자 편향' 사례에서, 잘못된 판단과 올바른 판단은 각각 무엇이었나요? 배운 내용을 바탕으로 적어보세요.

4. 설문에서 "우리 학교 급식이 맛없다고 생각하지 않나요?"라는 질문이 문제인 이유는 무엇인가요? 중립적으로 바꾸면 어떻게 고칠 수 있을까요?

5. 좋은 설문의 다섯 가지 원칙 중에서, '한 번에 하나만 묻기'를 지키지 않으면 어떤 문제가 생길 수 있나요? 이중 질문 예를 하나 들어 설명해 보세요.

6. '데이터는 어디서 오는가' 단원을 배운 뒤, 일상에서 만난 설문이나 조사 중 '편향'이나 '유도 질문'이 의심되는 예가 있다면 적어보세요.
