# 모듈 5 수업자료: 시각화로 이야기하기

데이터를 차트로 표현하고, 올바르게 읽는 법을 배우는 모듈입니다.

| 레슨 | 제목 | 소요 시간 | 핵심 |
|------|------|----------|------|
| 5-1 | 차트 읽기 | 15분 | 차트 리터러시, 5단계 읽기, 막대/꺾은선/원형 |
| 5-2 | 올바른 차트 선택하기 | 20분 | 데이터 유형별 차트 매칭, 의사결정 가이드 |
| 5-3 | 나만의 차트 만들기 | 25분 | 차트 빌더, 집계, 미션 |
| 5-4 | 거짓말하는 차트 | 20분 | 4가지 속임수, 비판적 읽기 |

## 5-1 차트 읽기

### 도입
- 데이터를 효과적으로 전달하는 방법 = **차트(그래프)**.
- 차트를 **제대로 읽을 줄 아는 것**이 중요하다.
- **차트 리터러시(Chart Literacy)**: 차트를 정확하게 읽고, 해석하고, 비판적으로 평가하는 능력.

### 차트 읽기 5단계
1. **제목 확인** — 이 차트가 무엇에 대한 차트인지 파악.
2. **축 확인** — X축(가로), Y축(세로)이 각각 무엇을 나타내는지 확인.
3. **단위 확인** — 점수, 온도, 명 등 단위에 따라 해석이 달라짐.
4. **패턴 파악** — 최대·최소값, 증가/감소 추세 파악.
5. **비교하기** — 항목 간 차이나 관계 비교.

### 1. 막대 차트
- **용도**: 범주별 값의 크기를 비교할 때.
- **읽는 포인트**: 막대의 **높이(또는 길이)**가 값의 크기.
- **실습 예**: 과목별 평균 점수 (국어, 수학, 영어, 과학, 사회).

### 2. 꺾은선 차트
- **용도**: 시간에 따른 변화를 보여줄 때.
- **읽는 포인트**: 선의 **기울기**와 **추세(트렌드)**. 올라가면 증가, 내려가면 감소.
- **실습 예**: 2025년 서울 월별 평균 기온.

### 3. 원형 차트
- **용도**: 전체에서 각 부분이 차지하는 **비율** 표현.
- **주의**: 항목이 **5~6개 이하**일 때 효과적. 항목이 많으면 비교가 어려움.
- **실습 예**: 학생 200명의 등교 교통수단 (도보, 버스, 지하철, 자가용, 자전거).

### 세 가지 차트 비교 표
| 차트 유형 | 적합한 상황 | 읽는 포인트 |
|-----------|-------------|-------------|
| 막대 차트 | 항목별 크기 비교 | 막대의 높이 비교 |
| 꺾은선 차트 | 시간에 따른 변화 | 선의 기울기와 추세 |
| 원형 차트 | 비율/구성 표현 | 조각의 크기(비율) |

### 도전: 차트에서 이야기 만들기
- 구체적인 수치 언급 → 비교 포함 → 해석/시사점 제시 = **데이터 기반 스토리텔링**.

## 5-2 올바른 차트 선택하기

### 핵심 원칙
- **"데이터로 하고 싶은 이야기가 무엇인가?"**를 먼저 정하고, 그에 맞는 차트를 선택.

### 주요 차트 유형과 용도
| 차트 | 언제? | 데이터 유형 | 예시 |
|------|-------|-------------|------|
| 막대 차트 | 범주별 크기 비교 | 범주형(X) + 수치형(Y) | 과목별 성적, 반별 학생 수 |
| 꺾은선 차트 | 시간에 따른 변화 | 시간(X) + 수치형(Y) | 월별 기온, 연도별 인구 |
| 원형 차트 | 전체 대비 비율 | 범주형 + 비율 | 교통수단 비율, 예산 구성 |
| 산점도 | 두 수치 변수의 관계 | 수치형(X) + 수치형(Y) | 키와 몸무게, 공부 시간과 성적 |
| 히스토그램 | 수치 데이터의 분포 | 수치형 하나의 분포 | 시험 점수 분포, 키 분포 |

### 차트 선택 의사결정 가이드 (질문 순서)
1. 비교하려는 것이 **"시간에 따른 변화"**인가? → YES: 꺾은선 차트.
2. 전체에서 각 부분의 **"비율"**을 보고 싶은가? → YES: 원형 차트 (항목 5개 이하).
3. 두 수치형 변수 사이의 **"관계"**를 보고 싶은가? → YES: 산점도.
4. 하나의 수치형 변수의 **"분포"**를 보고 싶은가? → YES: 히스토그램.
5. 범주별 값의 **"크기"**를 비교하고 싶은가? → YES: 막대 차트.

### 데이터 유형과 차트 매칭 예시
- 2020~2025년 학교 급식 만족도 변화 → **꺾은선**
- 혈액형 비율 → **원형**
- 키와 50m 달리기 기록의 관계 → **산점도**
- 5개 반의 봉사활동 시간 비교 → **막대**
- 학생 200명의 수학 점수 분포 → **막대(히스토그램)** 또는 막대
- 동아리별 가입자 수 → **막대**

### 잘못된 차트 선택 (흔한 실수)
1. **시간 변화** 데이터를 원형 차트로 그리기 → 꺾은선이 적합.
2. **너무 많은 항목**(10개 이상)을 원형 차트에 넣기.
3. **두 변수의 관계**를 막대 차트로 그리기 → 산점도가 적합.

### 실습 3: 이 차트 선택, 괜찮을까?
- 10년간 서울 인구 변화를 원형 차트로 표현 → **문제 있음** (꺾은선 적합).
- 반별 학생 수를 막대 차트로 표현 → **적절**.
- 키와 몸무게 관계를 원형 차트로 표현 → **문제 있음** (산점도 적합).
- 좋아하는 계절 비율을 원형 차트로 표현 → **적절**.

## 5-3 나만의 차트 만들기

### 사용 데이터
- **200명 학생의 학교 설문 데이터**
- 항목: 학년, 교통수단, 수학점수, 급식만족도(1~5), 스크린타임(시간).

### 차트 빌더 옵션
- **차트 유형**: 막대, 가로 막대, 꺾은선, 원형, 산점도, 히스토그램.
- **X축 / Y축**: 데이터 열 선택 (범주형/수치형 표시).
- **집계 방법**: 평균, 합계, 개수, 최댓값, 최솟값 (범주+수치 조합 시).
- **차트 제목**: 선택 입력.

### 차트 만들기 미션
1. **학년별 평균 수학 점수 비교** — 막대, X: 학년, Y: 수학점수, 집계: 평균.
2. **교통수단별 학생 수 비율** — 원형, 데이터 열: 교통수단.
3. **수학 점수 분포 확인** — 히스토그램, 데이터 열: 수학점수.
4. **스크린타임과 수학 점수의 관계** — 산점도, X: 스크린타임, Y: 수학점수.

### 정리
- 같은 데이터라도 **차트 유형**을 바꾸면 강조되는 정보가 달라진다.
- **어떤 이야기를 하고 싶은지**에 따라 차트를 선택하자.

## 5-4 거짓말하는 차트

### 도입
- 차트는 **설득의 도구**로도 쓰일 수 있다. 의도적·무의식적 **왜곡**에 주의.
- **비판적 시각**이 필수.

### 차트 속임수 4가지

#### 1. 잘린 Y축 (Truncated Y-Axis)
- **기법**: Y축을 0이 아닌 데이터 범위에 맞춰 잘라냄.
- **효과**: 작은 차이가 매우 큰 것처럼 보임.
- **예**: 후보별 지지율 47.2% vs 44.1% (차이 3.1%p)를 Y축 43~48%로 잘라 압도적 차이처럼 보이게 함.
- **발견법**: **Y축이 0에서 시작하는지** 확인.

#### 2. 체리 피킹 (Cherry-Picking)
- **기법**: 유리한 기간·구간의 데이터만 선택적으로 제시.
- **효과**: 전체 추세와 반대되는 결론 유도 가능.
- **예**: 7~10월 매출만 보여주면 "성장"처럼 보이지만, 1~6월 하락 포함 시 연초 대비 감소일 수 있음.
- **발견법**: **시작점·끝점·전체 기간**이 표시되었는지 확인.

#### 3. 시각적 왜곡 (Visual Distortion)
- **기법**: 특정 항목을 끌어내기(pull out), 색상·3D로 강조해 실제보다 크게/작게 보이게 함.
- **예**: 시장 점유율 28%인 "우리 회사"를 조각을 끌어내고 진한 색으로 가장 커 보이게 표현. 실제 1위는 32%인 경쟁사 A.
- **발견법**: **실제 수치(%)**를 직접 비교. 시각적 크기에만 의존하지 말 것.

#### 4. 거짓 인과관계 (Spurious Correlation)
- **기법**: 두 변수를 같은 차트에 겹쳐 놓아, 인과관계가 있는 것처럼 보이게 함.
- **핵심**: **상관관계 ≠ 인과관계**. 함께 변한다고 A가 B의 원인은 아님.
- **예**: 아이스크림 판매량과 익사 사고 — 둘 다 **기온**이라는 제3의 변수 영향. "제3변수 문제".
- **발견법**: "A가 증가할 때 B도 증가"는 상관일 뿐, 원인-결과로 해석하지 말 것.

### 차트를 볼 때 확인할 체크리스트 (6가지)
1. **Y축이 0에서 시작하는가?** — 잘린 Y축 과장 방지.
2. **전체 기간의 데이터가 표시되어 있는가?** — 체리 피킹 방지.
3. **3D·특수 시각 강조가 있는가?** — 시각적 왜곡 의심.
4. **상관관계를 인과관계로 해석하고 있지 않은가?**
5. **축의 눈금 간격이 일정한가?** — 불균등 간격은 변화율 왜곡 가능.
6. **출처와 조사 방법이 명시되어 있는가?** — 신뢰성 확인.

### 핵심 원칙
- "이 차트가 나에게 **무엇을 설득하려 하는가?**"를 자문.
- 차트 메시지를 그대로 받아들이지 말고, **데이터 자체**를 먼저 확인하는 습관.

## 모듈 5 종합: 이번에 배운 것

- **5-1**: 차트 읽기 5단계(제목→축→단위→패턴→비교), 막대/꺾은선/원형의 용도와 읽는 법, 데이터 기반 스토리텔링.
- **5-2**: 5가지 차트 유형과 용도, 의사결정 가이드, 데이터 특성에 맞는 차트 매칭, 잘못된 차트 선택 비판.
- **5-3**: 데이터 열·차트 유형·집계 방법 선택으로 차트 만들기, 같은 데이터의 다양한 표현.
- **5-4**: 잘린 Y축, 체리 피킹, 시각적 왜곡, 거짓 인과관계 식별, 비판적 읽기 체크리스트.

## 레슨 과제

1. 차트를 읽을 때 확인하는 '5단계'(제목, 축, 단위, 패턴, 비교) 중에서, 왜 '단위 확인'이 중요한지 한 가지 이유를 적어보세요.

2. '시간에 따른 변화', '전체 대비 비율', '범주별 크기 비교' 중 하나를 골라, 그에 가장 적합한 차트 유형을 쓰고, 그렇게 생각한 이유를 한 문장으로 설명해 보세요.

3. 같은 데이터(예: 교통수단 이용 현황)를 막대 차트로 그렸을 때와 원형 차트로 그렸을 때, 강조되는 정보가 어떻게 달라질 수 있나요?

4. '거짓말하는 차트'에서 배운 네 가지 속임수(잘린 Y축, 체리 피킹, 시각적 왜곡, 거짓 인과관계) 중 하나를 골라, 뉴스나 광고에서 비슷한 예가 있을 수 있는 상황을 상상해 적어보세요.

5. 차트를 볼 때 "Y축이 0에서 시작하는가?"를 확인하는 이유는 무엇인가요? 잘린 Y축이 있으면 어떤 오해가 생길 수 있나요?

6. '시각화로 이야기하기' 단원을 배운 뒤, 일상에서 본 차트 중 '한눈에 읽기 좋았던 차트' 또는 '의심이 든 차트'가 있다면 그 이유와 함께 적어보세요.
