# 챕터 2 수업자료: 질문을 설계하라

막연한 궁금증을 데이터로 답할 수 있는 질문으로 변환하는 챕터입니다.

| 레슨 | 제목 | 소요 시간 | 핵심 |
|------|------|----------|------|
| 2-1 | 궁금증에서 질문으로 | 20분 | 3단계 변환, 좋은 질문 4조건 |
| 2-2 | 상황카드로 질문 만들기 | 25분 | 상황→변수→질문, 6가지 영역 |
| 2-3 | 좋은 질문의 조건 | 15분 | 4가지 체크포인트, 개선 3팁 |

---

## 2-1 궁금증에서 질문으로

### 도입
- 데이터 분석의 시작은 **좋은 질문**. 막연한 궁금증을 데이터로 답할 수 있는 질문으로 바꾸는 연습.

### 질문의 3단계 변환
1. **막연한 궁금증** — 일상에서 느끼는 모호한 느낌
2. **구체적 호기심** — 변수와 관계를 포함한 질문
3. **데이터 질문** — 측정 가능·분석 가능한 질문

### 좋은 데이터 질문의 4가지 조건
1. **측정 가능한 변수** 포함 (스마트폰 사용 시간, 수면 시간 등)
2. **분석 대상** 명확 (고등학생, 서울 시민 등)
3. **데이터를 모으면 답할 수 있음** (주관적 판단이 아닌 사실 확인)
4. **중립적** (결론을 미리 정하지 않음)

### 변환 예시
- "요즘 학생들이 잠을 못 자는 것 같아" → "스마트폰 사용 시간이 수면 시간에 영향을 줄까?" → "고등학생의 일일 스마트폰 사용 시간과 수면 시간 사이에 음의 상관관계가 있는가?"

### 나쁜 질문 vs 좋은 질문
- "스마트폰은 나쁜가?" → 주관적 판단 질문 → "일일 스마트폰 사용 시간과 학업 성취도 사이에 관계가 있는가?"
- "왜 학생들은 공부를 안 할까?" → '왜' 인과 질문 → "일일 공부 시간이 적은 학생(2시간 미만)은 어떤 특징이 있는가?"
- "날씨가 따뜻해지고 있나?" → 시간/장소/기준 모호 → "2015~2025년 서울의 연평균 기온이 상승 추세를 보이는가?"

## 2-2 상황카드로 질문 만들기

### 좋은 데이터 질문 만드는 순서
1. 상황 속 궁금증 찾기
2. 측정 가능한 변수 떠올리기
3. 변수들 사이의 관계로 질문 만들기

### 상황카드 6가지 영역 (예시)
| 영역 | 궁금증 예 | 측정 가능한 변수 예 |
|------|-----------|---------------------|
| 학교생활 | 점심시간이 짧아 급식을 다 못 먹는 학생이 많다 | 대기 시간, 식사 시간, 남기는 양, 만족도 |
| 환경 | 여름이 점점 더 더워진다 | 연평균 기온, 최고 기온, 폭염·열대야 일수 |
| 건강 | 스마트폰을 많이 하면 눈이 나빠진다 | 스마트폰 사용 시간, 시력, 나이, 야외 활동 시간 |
| 경제 | 용돈을 많이 받는 친구가 더 행복해 보인다 | 용돈 액수, 생활 만족도, 저축·소비 항목 |
| 문화 | 좋아하는 음악 장르가 나이에 따라 다르다 | 나이, 선호 장르, 청취 시간, 유료 구독 여부 |
| 기술 | AI를 쓰면 공부를 덜 하게 된다 | AI 사용 빈도, 자기주도 학습 시간, 성적, 과제 완성도 |

### 예시: 급식 → 데이터 질문
- "점심시간이 짧아서 급식을 다 못 먹는다" → "배식 대기 시간이 실제 식사 시간에 영향을 주는가?" → "배식 대기 시간이 10분 이상인 학생의 식사 잔반량이 대기 5분 미만 학생보다 유의하게 많은가?"

## 2-3 좋은 질문의 조건

### 질문 품질 4가지 체크포인트
1. **측정 가능한 변수 포함** (📏)
2. **분석 대상 명확** (🎯)
3. **데이터로 답 가능** (📊)
4. **중립적 질문** (⚖️)

4가지 모두 충족하면 좋은 데이터 질문. 하나라도 부족하면 다듬을 부분이 있음.

### 질문 개선 3가지 팁
| 팁 | 나쁜 예 | 좋은 예 |
|----|--------|--------|
| "많이", "적게" → 구체적 숫자 | "공부를 많이 하면" | "일 평균 4시간 이상 공부하면" |
| "~해야 하는가" → "~인가" | "운동을 해야 하는가?" | "운동 빈도와 체력 지수 사이에 상관관계가 있는가?" |
| 비교 기준 명시 | "성적이 높은가?" | "A반의 평균 성적이 B반보다 높은가?" |

### 평가 예시
- "스마트폰 사용 3시간 이상인 고등학생의 수면 시간이 3시간 미만 학생보다 짧은가?" → 4/4 좋은 질문
- "학생들이 운동을 해야 하는가?" → 0/4 가치 판단 질문
- "서울 1월 평균 기온은 2015~2025 상승 추세인가?" → 4/4 완벽한 질문
- "공부를 많이 하면 성적이 오를까?" → "많이" 모호 → 구체화 필요

## 챕터 2 종합: 이번에 배운 것

- **2-1**: 3단계 변환(막연한 궁금증→구체적 호기심→데이터 질문), 좋은 질문 4조건, 나쁜 질문 개선 방법.
- **2-2**: 다양한 상황에서 데이터 질문의 씨앗 찾기, 상황→변수 식별→질문 구조화, 6가지 영역 예시.
- **2-3**: 질문 품질 4가지 체크포인트, 실전 질문 평가, 모호한 질문을 구체적으로 바꾸는 3가지 팁.

---

## 레슨 과제

1. '좋은 데이터 질문'의 네 가지 조건(측정 가능한 변수, 분석 대상 명확, 데이터로 답 가능, 중립적)을 떠올리며, "학생들이 스마트폰을 너무 많이 한다"는 말을 데이터 질문 한 개로 바꿔 보세요.

2. 막연한 궁금증 → 구체적 호기심 → 데이터 질문의 3단계 변환을 적용해, 요즘 관심 있는 주제로 데이터 질문 하나를 만들어 보세요.

3. "운동을 해야 하는가?"라는 질문은 데이터 질문으로 보기 어렵습니다. 왜 그런지, 그리고 어떻게 바꾸면 좋을지 적어보세요.

4. 상황카드처럼 '학교생활', '건강', '환경' 중 하나를 골라, 그 영역에서 측정 가능한 변수 두 가지를 쓰고, 그 변수들을 이용한 데이터 질문 한 개를 만드세요.

5. 질문 품질의 네 가지 체크포인트(측정 가능, 분석 대상 명확, 데이터로 답 가능, 중립적)로 다음 질문을 평가해 보세요. "우리 반 학생들의 평균 키가 전국 평균보다 큰가?"

6. '질문을 설계하라' 단원을 배운 뒤, 뉴스나 SNS에서 본 '데이터로 답할 수 있는 질문'과 '닿기 어려운 질문' 예를 각각 하나씩 들어보세요.
