## 6-1 상관관계 놀이터

### 상관계수(r)
- 두 변수 사이 관계를 숫자로 나타낸 것. -1–+1 사이 값.
- r = +1: 완벽한 양의 상관 (X↑ → Y↑)
- r = 0: 상관 없음
- r = -1: 완벽한 음의 상관 (X↑ → Y↓)

### r값 해석 기준 (대략)
| |r| 범위 | 해석 |
|--------|------|
| > 0.7 | 강한 상관 |
| 0.4 ~ 0.7 | 보통 상관 |
| 0.2 ~ 0.4 | 약한 상관 |
| ≤ 0.2 | 거의 없음 |
| = 0 | 무상관 |

### 실습 포인트
- 양의 상관 데이터에서 한 점을 크게 벗어나게 하면 r이 어떻게 변하는지 관찰.
- 수평선 형태로 점을 찍으면 r ≈ 0.
- 완벽한 대각선이면 r ≈ 1.

### 유의
- 상관계수가 높다고 반드시 의미 있는 관계는 아님. 상관관계와 인과관계는 다름 (다음 레슨).

---

## 6-2 상관 ≠ 인과

### 핵심 원칙
- "상관관계는 인과관계가 아니다" — 두 변수가 함께 변한다고 하나가 다른 하나를 유발하는 것은 아님.
- 상관관계: X가 변할 때 Y도 함께 변하는 패턴 (관찰만으로 알 수 있음).
- 인과관계: X가 Y를 실제로 유발함 (원인→결과). 실험이나 추가 증거가 필요.

### 숨겨진 제3변수(교란변수)
- 두 변수가 함께 변하는 이유가 다른 변수 때문인 경우가 많음.
- 예: 아이스크림 판매량 ↑, 익사 사고 ↑ → 상관은 있으나 인과 아님. 둘 다 기온(여름)이라는 제3변수의 영향.

### 상관 vs 인과 퀴즈 예시
- "아이스크림 판매량↑ → 익사↑" → 상관일 뿐 (제3변수: 기온)
- "흡연량↑ → 폐암↑" → 인과 (연구로 입증)
- "신발 크기↑ → 수학 점수↑" → 상관일 뿐 (제3변수: 나이)
- "해적 수↓ → 지구 온도↑" → 상관일 뿐 (시간의 흐름)
- "백신 접종률↑ → 전염병↓" → 인과 (RCT로 입증)

### 심슨의 역설
- 전체 데이터에서 보이는 추세가, 하위 그룹으로 나누면 반대가 되는 현상. 그룹을 나눠서 다시 확인하는 습관이 중요.

### 인과관계를 밝히려면?
1. 무작위 대조 시험(RCT) — 가장 강력 (의약품 임상시험)
2. 교란변수 통제 — 제3변수 영향 제거
3. 시간적 선후관계 — 원인이 결과보다 먼저 발생
4. 용량-반응 관계 — 원인이 많을수록 결과도 커지는지

## 6-3 비교 분석

### 평균만으로는 부족
- "A반 평균이 B반보다 높다"고 해서 A반이 더 잘한 것만은 아님. 분포를 함께 봐야 올바른 비교 가능.

### 평균이 같아도 분포는 다르다
- A반: 고른 분포 (평균 72.6, 표준편차 4.5) / B반: 양극화 (평균 73.5, 표준편차 21.2). 평균의 함정: "우리 반 평균 72점"이라고 해도 모두 70점대인 반과 40·100점이 반반인 반이 같은 평균. 분산(퍼진 정도)를 반드시 확인.

### 박스 플롯으로 비교
- 그룹별 분포(중앙값, 사분위수, 이상치)를 한눈에 비교. 예: 운동 O vs 운동 X 그룹의 수면 시간 — 운동 O 그룹이 수면 시간이 더 긴 경향 등.

### 그룹 비교 체크리스트
1. 평균(대표값) 비교
2. 분포(박스 플롯 또는 히스토그램) 확인
3. 표본 크기 확인 (각 그룹 n?)
4. 차이가 "우연"인지 "의미 있는" 차이인지 판단

## 6-4 분석 결과 검증 5단계

### 검증의 중요성
- "결과가 나왔으니 끝"이 아니라 "이 결과를 믿어도 되는가?"를 확인하는 것이 데이터 리터러시의 핵심.

### 검증 5단계
| Step | 제목 | 확인 질문 | 세부 |
|------|------|-----------|------|
| 1 | 숫자 확인 | 데이터 크기·통계값이 맞는가? | 표본 수, 평균, 상관계수 등 |
| 2 | 시각화 확인 | 차트가 데이터를 정확히 반영하는가? | 축 범위, 차트 유형, 레이블 |
| 3 | 상식 검증 | 결과가 상식적으로 말이 되는가? | 도메인 지식으로 합리성 판단 |
| 4 | 민감도 분석 | 조건을 바꿔도 결론이 유지되는가? | 이상치 제거, 기간 변경 등 |
| 5 | 대안 분석 | 다른 방법으로도 같은 결론인가? | 다른 분석, 교란변수 통제 후 |

- 5단계를 모두 통과하면 신뢰할 만한 결과. 하나라도 실패하면 추가 조사·주의 필요. 특히 3단계(상식)와 5단계(대안)가 중요.

### 실전 시나리오 예
- "스마트폰 사용↑ → 성적↓ (r=-0.82)" → 숫자·시각화는 OK, 인과 주장은 위험, 이상치 제거 시 r 변동 큼, 학습 시간 통제 시 상관 약해짐 → 주의 필요.
- "운동일수↑ → 생활 만족도↑ (r=0.45)" → 5단계 모두 통과, 신뢰할 만함.

## 챕터 6 종합: 이번에 배운 것

- 6-1: 상관계수 r의 의미(-1–+1), r값 해석 기준, 데이터 분포가 r에 미치는 영향.
- 6-2: 상관 ≠ 인과, 제3변수(교란변수), 심슨의 역설, 인과관계를 밝히려면 실험이 필요.
- 6-3: 그룹별 평균 비교, 평균이 같아도 분포가 다를 수 있음, 박스 플롯, 그룹 비교 시 확인할 4가지.
- 6-4: 분석 결과 검증의 중요성, 5단계(숫자→시각화→상식→민감도→대안), 실전 시나리오에 적용.

## 레슨 과제

1. 상관계수 r이 -0.8일 때와 0.3일 때, 두 변수 사이의 관계를 어떻게 다르게 해석할 수 있나요?

2. "아이스크림 판매량이 늘면 익사 사고도 늘어난다"는 말은 상관관계일 뿐 인과관계가 아닐 수 있습니다. 그럴 수 있는 이유(제3변수)를 배운 내용을 바탕으로 적어보세요.

3. 두 그룹을 비교할 때 '평균'만 보면 부족한 이유는 무엇인가요? 평균이 같아도 분포가 다를 수 있는 예를 들어 설명해 보세요.

4. 분석 결과를 검증하는 5단계(숫자 확인, 시각화 확인, 상식 검증, 민감도 분석, 대안 분석) 중에서, '상식 검증'이 중요한 이유를 한 문장으로 적어보세요.

5. "스마트폰 사용 시간이 길수록 성적이 낮다"는 분석 결과를 믿기 전에, 우리가 확인해 보아야 할 것(예: 표본, 인과 해석, 제3변수 등)을 두 가지 이상 써 보세요.

6. '분석하고 검증하기' 단원을 배운 뒤, 뉴스나 SNS에서 본 '상관을 인과처럼 말한 경우' 또는 '검증이 부족해 보인 분석' 예가 있다면 적어보세요.
